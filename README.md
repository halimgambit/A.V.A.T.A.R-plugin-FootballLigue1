# Football Ligue1
le classement de la ligue 1

This plugin is an add-on for the [A.V.A.T.A.R](https://avatar-home-automation.github.io/docs) framework.

🎯 Usage
Commandes :

quel est le classement de ligue 1
donne moi le classement de ligue 1

Multi-room
The FootballLigue1 plugin is fully multi-room.

Multi-language
The FootballLigue1 plugin relies solely on the system's available languages.
